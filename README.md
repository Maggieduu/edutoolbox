# EduToolbox

AI 赋能的教学工具平台 | AI-Powered Tools for  Educators

## 🎯 简介

Edu Toolbox 为教育工作者提供一系列有趣的互动工具，帮助学生在游戏中学习。

## ✨ 功能

### 🎮 游戏

- 🎮 五子棋 - 复习同义词反义词
- 🔍 汉字大侦探 - 词语搜索游戏
- 🃏 生词翻翻看 - 卡牌记忆游戏
- 🐍 贪吃蛇 - 吃掉错误单词
- 📝 疯狂的句子 - 组合正确句子

### 🛠️ 工具

- 🖨️ 汉字大侦探生成器 - 生成词语搜索工作纸
- 🎯 随机点名器 - 随机抽取学生

### 🤖 AI 助手 (即将上线)

- 作文批改助手
- 造句生成器
- 阅读理解生成
- 情境对话生成

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS (部分页面)
- 无需后端，纯静态部署
